# Freely-hostable research papers

Papers cited on the wiki for which a **legal open copy exists** (author accepted-manuscript, NBER/working-paper, institutional repository, or public domain) that we could mirror on our own infrastructure if we ever want citation stability against link-rot. This is a **tracking list for a future decision to self-host** — nothing here has been mirrored yet.

Hosting basis legend: **WP** = freely distributable working paper (NBER/RFF/SSRN author copy); **AM** = author accepted-manuscript posted on the author's own site (green OA — generally mirrorable as the accepted MS, not the publisher PDF); **PD** = public domain; **OA** = gold open-access (CC licence — mirrorable). Where the version of record is paywalled, the open URL is the mirrorable version.

## How to grow this list
Each VERIFY-sweep shift, when a cited paper turns out to have an author/working-paper/OA copy, add a row here. When the version of record is open-access (CC), mark it OA and it can be mirrored verbatim; for AM/WP copies, mirror the open version and cite the DOI of record.

**Not freely hostable (noted so we don't re-check):** IMF publications (e.g. Country Report 17/70, Belgium — cited on allowance-for-corporate-equity) — IMF terms (Oct 2024) require written permission via copyright@imf.org to republish; cite the canonical URL + a Wayback snapshot instead, or email IMF for permission if self-hosting a specific report matters. Tiebout (1956, *JPE*) — still under Chicago copyright; a teaching copy circulates (fbaum.unc.edu) but is not licensed for mirroring. Albouy, Ehrlich & Shin (2018, *REStat*) — paywalled, no clear author OA copy found yet.
